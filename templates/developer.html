Project Details

This project uses Support Vector Machine (SVM) classifier trained on 4,920 medical records with 132 symptoms to predict among 41 different diseases. The system achieves 100% accuracy on the test dataset and provides comprehensive health recommendations including precautions, medications, diet plans, and workout suggestions.
